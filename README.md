# AI-Based FIR Generation & Legal Section Classification

## 📌 Overview
This project automates FIR (First Information Report) generation and predicts relevant legal sections using NLP and Machine Learning.

## 🚀 Features
- FIR generation from complaint text
- Legal section prediction
- Tamil + English input support
- Structured FIR output
- Optional speech input support

## 🧠 Tech Stack
- Python
- NLP (BERT / Transformers)
- Machine Learning
- Flask / Streamlit (if used)

## 📂 Project Structure
- fir_app.py → Main application
- fir_generator.py → FIR generation logic
- model_definition.py → ML-model
- db_manager.py → Database handling
- text_processor.py → NLP preprocessing
